## 구상
* 염두에 두기
    * 레이어드 아키텍쳐 설계 필요
    * 일부 기능들은 다른 프로젝트에서도 사용할 수 있을 정도로 충분히 추상화 하기

* UIManager
    * UI 정렬순서, 계층 관리 
    * UI생성
    * UI삭제

* UIBase - UI들이 상속
    * UI 보이기
    * UI 숨김
    * UI 제거
    * 이벤트 - 파괴,보이기,숨김시

* UI 종류
    * HudUI
        * 스크린 영역에 표시, 플레이어가 상호작용할 수 없음
        * 예시
            * 체력바
            * 'X키를 눌러 아이템 획득' 안내 메시지
    * MenuUI
        * 화면에 고정됨, 클릭이나 버튼으로 상호작용할 수 있는 UI
        * 예시
            * 인벤토리 UI
            * 일시정지 UI
    * WorldSpaceUI
        * 스크린 영역이 아닌 캐릭터와 같은 월드공간에 표시되는 UI
        * 예시
            * 데미지 숫자 표시
