### SceneLoader
* 씬 이동 관리자가 아님, 씬을 한번 로드할때 사용하는 일회성 객체
* 함수
    * LoadScene() 씬을 로드한다
    * AddContext<T>(T context) 로드할 씬의 LifeTimeScope에 반영할 Context를 추가한다
* 활용 예
    * `AddContext<StageContext>(data)` - 로드할 씬에 필요한 컨텍스트를 추가
    * `LoadScene()` - 최종적으로 씬을 로드한다

### StageLoader활용한 스테이지 선택창을 통한 스테이지 씬 이동 예시 시퀀스 다이어그램
```mermaid
sequenceDiagram
    autonumber
    participant StageSelector
    participant SceneLoader
    participant StageManager
    activate StageSelector
    note over StageSelector: 스테이지 선택
    StageSelector ->> SceneLoader : AddContext<StageContext>()
    StageSelector ->> SceneLoader : LoadScene()
    deactivate StageSelector
    activate SceneLoader
    SceneLoader ->> SceneLoader : LifetimeScope.Enqueue를 통해 직전에 AddContext로 추가한 컨텍스트들 예약 등록
    SceneLoader -->> StageManager : 씬 로드
    deactivate SceneLoader
    note over StageManager: 로드 전 추가된 컨텍스트를 활용해 씬 초기화
```
