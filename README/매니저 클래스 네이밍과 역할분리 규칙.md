# Manager, Service, Helper

> 일단은 아래 세가지 역할정도로만 구분

- `Manager` 계열 클래스
	- 내부에 상태를 가질 수 있음
	- 서비스,시스템 로직 수행, 게임적인 영역, 관리 등
	- 예시
		- `InputManager` - 키 입력을 받고 입력 이벤트를 쏜다
		- `PauseManager` - 일시정지 요청들을 관리하고 요청 갯수에 따라 일시정지를 수행한다
- `Service` 계열 클래스
	- 내부에 상태를 가질 수 있음
	- 기획적인 비즈니스 로직, 기능과 계산 위주
	- 예시 
		- `PlayerLevelService` - 경험치를 더하고 경험치가 모두 쌓이면 레벨이 올라간다는 비즈니스 로직을 가진다
- `Helper` 계열 클래스
	- 내부에 어떠한 상태도 가질 수 없음
	- 오직 로직만 제공
	- 예시
		- `MathHelper` - 수학 관련 연산 함수 제공
