# 개요
> 몬스터 AI를 위한 행동트리의 구조 설계입니다

# 설계

 
```mermaid
classDiagram
    class AiBTContext["AiBTContext<br>AI 행동트리 컨텍스트"]{
        +Self : EntityBase
        +Target : EntityBase
        ... 그 외 행동트리 작동에 필요한 정보
    }

    class AiBTBase["AiBTBase<br>행동트리 베이스"]{
        <<abstract>>
        
        +Evaluate(AiBTContext context) void
    }

    class ConcreteAiBT["ConcreteAiBT : ScriptableObject<br>행동트리 상세 구현 및 구성"]{
        
    }

    class AIComponent["AIComponent : Monobehaviour<br>AI 컴포넌트"]{
        -Context : AiBTContext
        -AiBT : AiBTBase
        +SetBT(AiBTBase aiBT)
    }
    AiBTContext <-- AIComponent : AiBTBase에 전달
    AiBTBase <-- AIComponent : Update()에서 실행
    AiBTBase <|-- ConcreteAiBT : 상속 후 실제 행동트리 구현
```
- `AiBTContext`
- `AiBTBase`
    - 행동 AI 행동트리 베이스 클래스
- `ConcreteAiBT`
    - 에디터에서 스크립터블 오브젝트를 생성해 스크립터블 오브젝트의 필드로 수치 편집 가능
        - 예: 적 감지 거리
- `AIComponent`
  - `EntityComponent`의 일종
