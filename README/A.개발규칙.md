## 외부에서 임포트한 에셋,리소스 관리 규칙
* Assets/ExternalAsset 폴더에 배치하는것을 기본으로 한다
* 외부 에셋의 리소스는 경우에 따라 사본을 만들어 사용한다
    * 변경/사용자 정의 설정 확률이 높은 리소스일수록 사본을 만들어 사용한다
    * 예를들면
        * 텍스쳐, 프리펩
            * 원본 쓰지 말고 사본 만들어 사용
        * 스크립터블 오브젝트, 머테리얼
            * 경우에 따라 적절히 판단, 수정이 필요한 경우 사본 만들기
        * 스크립트
            * 원본 그대로 사용
